# Guideline

## Build Guideline

1. Pull the code from GitHub.

2. Use command line tool to go to download directory.

   ```
   cd music-player
   ```

3. Update all pods.

   ```
   pod update
   ```

4. Click `music-player.xcworkspace` to open the workspace.

5. Build the app through XCode.

6. Enjoy :beers:.
